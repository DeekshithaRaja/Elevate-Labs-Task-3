SHORT DESCRIPTION ABOUT THE PROJECT:

This project is a REST API built using Node.js and Express to manage a list of books. We created a backend server and implemented different API endpoints to perform CRUD operations such as retrieving, adding, updating, and deleting books. The data is stored in memory using an array, without using any database. We used middleware to handle JSON data and process client requests. The API was tested using Postman to send HTTP requests and verify responses. Overall, this project demonstrates the fundamentals of REST APIs, routing, and backend development.
